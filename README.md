#🔮  灵塔纪（AI Tarot App）

一个基于 AI 的塔罗解读应用，帮助用户在不确定中获得结构化思考与理性建议。

👉 在线输入问题 → 抽取三张塔罗牌 → 获取 AI 个性化解读

---

## ✨ 产品亮点

- 🎴 **三张塔罗抽牌机制**
  - 随机抽取 + 正逆位判断
- 🧠 **AI 解读能力（Gemini）**
  - 结合问题 + 牌面生成自然语言解读
- 📚 **历史记录功能**
  - 自动保存用户每次占卜
  - 支持点击回看
- ⚡ **即时交互体验**
  - 无刷新获取 AI 结果
- 🧩 **结构化输出**
  - 整体解读 / 逐张分析 / 行动建议

## 🚀 技术栈

- **前端**：Next.js 15（App Router）
- **语言**：TypeScript
- **UI**：原生 CSS（轻量设计）
- **数据存储**：localStorage（历史记录）
- **AI 模型**：Google Gemini（gemini-2.5-flash）
- **API 架构**：Next.js API Route

---

## 🧠 核心功能说明

### 1️⃣ 抽牌逻辑
- 从 22 张塔罗牌中随机抽取 3 张
- 随机生成正位 / 逆位
- 前端完成随机逻辑（无需后端）

---

### 2️⃣ AI 解读流程

用户输入问题后：

1. 前端生成抽牌结果
2. 调用 `/api/tarot-reading`
3. 后端构造 Prompt（包含问题 + 牌面）
4. 调用 Gemini API
5. 返回结构化解读

---

### 3️⃣ 历史记录设计

- 使用 `localStorage` 持久化数据
- 每条记录包含：
  - 问题
  - 抽牌结果
  - AI 解读
  - 时间
- 支持：
  - 查看历史
  - 点击恢复
  - 清空记录
